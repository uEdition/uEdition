# Recipes

This area contains a collection recipies for solving specific problems.

- [Mixed-content XML conversion](mixed-content-xml): The μEdition does not support mixed-content TEI XML. This recipe
  provides a simple conversion script to convert mixed-content TEI, in order to be able to use it in the μEdition.
- [Converting TEI block elements](tei-block-content-wrap): The μEdition does not support text content that is contained
  directly within a TEI element configured as a block element. This recipe provides a simple conversion script to
  wrap such TEI content, in order to be able to use it in the μEdition.
