# Editing a page

The next step, shown in the following video, is to edit a single page.

:::{room3b-video} /uedition/tutorial/edit-page/en
:::

## Individual steps

1. On the left, in the file browser, select the file to edit. In this example we will select the file {file}`index.md`
   in the {file}`en` folder, which contains the content of the starting page.
2. Simply use the text editor to edit the file as desired. By default the μEdition uses Markdown as its text format.
   Markdown is a relatively simple format for marking up text. In the next step of the tutorial we will look at its
   markup functionality in a bit more detail, but for now it is sufficient to just add some text. To create paragraphs,
   split text with an empty line.
3. Save the changes by pressing {kbd}`Ctrl+s` (Mac: {kbd}`Command+s`) or by clicking on the Save icon in the top-left
   corner.
4. The μEdition will detect the changes and automatically update the HTML page.
