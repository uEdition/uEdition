# Metadaten bearbeiten

Der Metadateneditor erlaubt ein direktes Bearbeiten des [TEI Headers](https://tei-c.org/release/doc/tei-p5-doc/en/html/HD.html).
Folgendes Video zeigt, wie der TEI Header bearbeitet werden kann:

:::{room3b-video} /uedition/tutorial/tei/metadata/de
:::

## Einzelschritte

1. Auf die {guilabel}`+` Schaltfläche klicken, um ein neues TEI Element anzulegen.
2. In das {guilabel}`Tag` Feld den Namen des neuen Elements eingeben: `tei:fileDesc`.
3. Das neue TEI Element auswählen und erneut auf die {guilabel}`+` Schaltfläche klicken, um ein neues TEI Element
   anzulegen.
4. In das {guilabel}`Tag` Feld den Namen des neuen Elements eingeben: `tei:titleStmt`.
5. Das neue TEI Element auswählen und erneut auf die {guilabel}`+` Schaltfläche klicken, um ein neues TEI Element
   anzulegen.
6. In das {guilabel}`Tag` Feld den Namen des neuen Elements eingeben: `tei:title`.
7. In das {guilabel}`Text` Feld den Textinhalt des neuen Elements eingeben: "Titel der Seite"
8. Auf den Speichern Icon klicken oder die Tastenkombination {kbd}`Strg+s` (Mac: {kbd:`Command+s`}) drücken.
