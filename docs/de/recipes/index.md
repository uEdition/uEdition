# Rezepte

Dieser Bereich bietet einen Sammlung an Rezepten für spezifische Problemfälle.

- [Mixed-content XML Konvertierung](mixed-content-xml): Die μEdition untertützt kein mixed-content TEI XML. Dieses
  Rezept bietet ein einfaches Konvertierungsskript um mixed-content TEI umzuwandeln, damit es in der μEdition genutzt
  werden kann.
- [TEI Blockinhalte umwandeln](tei-block-content-wrap): Die μEdition untertützt keine Textinhalte, die direkt in einem
  als Blockelement konfiguriertem TEI Element stehen. Dieses Rezept bietet ein einfaches Konvertierungsscript, um
  derartige Inhalte mit einem zusätzlichen nicht-Blockelement zu versehen.
